# productivity-skills

Agent skills for personal productivity — calendar, focus modes, meeting prep, communication compression, article editing, Obsidian, and subscription optimization.

---

## Installation

```bash
npx skills add https://github.com/sabarish-duvvuru/productivity-skills
```

---

## Available Skills

### Calendar & Meetings

| Skill | Purpose |
|-------|---------|
| **meeting-check** | Fast meeting context — calendar, recaps, transcripts |
| **hl-calendar** | Headless Outlook Calendar automation |
| **hl-calendar-native** | Direct macOS Calendar.app bridge |
| **de-standup** | DE weekly stand-up system — dashboard, PPTX, updates |

### Focus & Environment

| Skill | Purpose |
|---------|---------|
| **focus-sync** | macOS Focus mode → shadow-prism beam sync |
| **focus-iterm-bridge** | iTerm2 profile switching, theming, badges |
| **init-tune** | Quick startup diagnostic and tuning |
| **hook-perf** | Audit and optimize hook performance |

### Communication

| Skill | Purpose |
|---------|---------|
| **caveman** | Ultra-compressed mode (~75% token reduction) |
| **caveman-compress** | Compress memory files into caveman format |
| **edit-article** | Restructure, clarify, tighten prose |
| **operator-patterns** | Learned interaction patterns — auto-applied |

### Knowledge & Notes

| Skill | Purpose |
|---------|---------|
| **bookmark-obsidian** | Save sessions as Obsidian bookmarks |
| **obsidian-vault** | Search, create, manage Obsidian notes |

### Subscriptions & Value

| Skill | Purpose |
|---------|---------|
| **hl-break-even** | Break-even analysis for subscriptions |
| **hl-value-ratio** | API cost vs actual spend ratios |
| **hl-cost-tracking** | Subscription usage and budget thresholds |
| **hl-live-api-testing** | Reverse-engineer APIs through live testing |
| **hl-auth-priority** | Credential resolution through priority chains |
